# AI - Modos Copiloto

Repositório desenvolvido como atividade prática do módulo **Inteligência Artificial Aplicada ao Desenvolvimento de Software**, parte do **Bootcamp Globant - Java & Spring Boot AI Developer**, promovido pela **[Digital Innovation One - DIO](https://www.dio.me/en)**.

O objetivo deste Lab Project é explorar o conceito de **copilotos de IA especializados**, criando diferentes modos de operação voltados para estudantes e desenvolvedores de software.

## Modos do Copiloto

### 📚 STUDY

Modo voltado para aprendizado e compreensão de conceitos.

**Objetivo:** ajudar o usuário a entender conceitos, intuições, trade-offs e aplicações práticas.

**Personalidade:** Cortana (Halo)



### 🗺️ PLAN

Modo responsável pelo planejamento de implementações.

**Objetivo:** elaborar planos incrementais, identificar riscos e definir estratégias antes da codificação.

**Personalidade:** TARS (Interstellar)



### 💻 CODE/EDIT

Modo focado em escrita, melhoria e refatoração de código.

**Objetivo:** produzir implementações claras, seguras e alinhadas às boas práticas.

**Personalidade:** Jarvis (Iron Man)



### 🔎 DEBUG

Modo especializado em investigação e resolução de problemas.

**Objetivo:** identificar causas-raiz e propor soluções baseadas em evidências.

**Personalidade:** EDI (Mass Effect)



### 🎯 PRACTICE

Modo voltado para desenvolvimento de habilidades por meio da prática.

**Objetivo:** fornecer exercícios, desafios e feedback progressivo.

**Personalidade:** Data (Star Trek)



## Fluxo dos Modos

```text
STUDY
   ↓
PLAN
   ↓
CODE/EDIT
   ↓
DEBUG
   ↓
PRACTICE
```

Esses modos representam diferentes etapas do processo de aprendizado e desenvolvimento de software, formando uma suíte de copilotos especializados.


## Tecnologias e Contexto

Os prompts foram projetados tendo como contexto principal:

* Java
* Spring Boot
* APIs REST
* SOLID
* Clean Architecture
* Streams e Collections
* JUnit
* Mockito
* MockMvc

Entretanto, os modos podem ser adaptados para outros domínios e tecnologias.

## Objetivo do Projeto

Demonstrar como diferentes perfis de copilotos podem auxiliar estudantes e desenvolvedores em atividades de aprendizado, planejamento, implementação, depuração e prática deliberada.
